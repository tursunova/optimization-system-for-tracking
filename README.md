## System for modeling of stochastic optimization algorithms for tracking

### HOWTO
1. Clone the project
2. Open project in Matlab
3. Run file `system.m`

If you don't have Matlab in your computer you can download standalone app through [link](https://drive.google.com/file/d/1767rPpCg2CY7Gm0_1Zx3Sh3TQ6bioOea/view?usp=sharing). In order to use all the functionality of the system, use the instructions mentioned above.
